空操作

ndoojs支持为app定义一个空的action，当通过pageID没有对定的action时会调用默认的action。

1.定义空操作

当action名为_emptyAction时会作为默认action被调用。

_n.app('home', function() {
  _emptyAction: function(params){
    var $content;
    $content = $('#content');
    $content.html('emptyAction');
  }
}

2.在页面中定义页面id。

<div id="content"></div>
<div class="Ldn" data-page-id="home/aaaa"></div>
