名称空间

在data-page-id或ndoo.init中支持以多个斜杠(/)的形式定义名称空间。

1. page id中设置名称空间，如 admin/user/add

<div id="scriptArea" data-page-id="admin/user/add">

2. 定义模块名称，使用.(点)分隔

_n.app('admin.user', {
  addAction: function(param){
    var $content;
    $content = $('#content');
    $content.html('namespace: admin module: user action: addAction');
  }
});
